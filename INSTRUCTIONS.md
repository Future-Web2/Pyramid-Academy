# Инструкция по запуску веб-сайта Pyramid Academy

Этот проект содержит дизайн-систему и демонстрационные версии главной страницы для **Pyramid Academy** (Zamonaviy Ta'lim Markazi) в Ташкенте.

В репозитории представлены два варианта дизайна:
1. **Стандартный фирменный стиль (Standard Branding)**:
   * Путь: `ui_kits/pyramid-academy-web/index.html`
   * Классический дизайн в глубоких темно-синих тонах (`--brand-navy`) с золотисто-янтарными акцентами (`--brand-amber`) и трекером успеваемости.
2. **Стиль Глассморфизма (Glassmorphism)**:
   * Путь: `ui_kits/pyramid-academy-glass/index.html`
   * Современный дизайн с полупрозрачными стеклянными панелями, размытием фона (`backdrop-filter`) и эффектом параллакса.

---

## Способы запуска сайта

Поскольку файлы используют Babel Standalone для компиляции JSX-компонентов прямо в браузере, при открытии файлов напрямую через двойной клик (по протоколу `file://`) современные браузеры заблокируют загрузку скриптов из-за политики безопасности CORS. Поэтому рекомендуется запускать локальный веб-сервер.

### Способ 1: С помощью Node.js (Рекомендуемый)

Мы добавили файл `package.json` с настроенным скриптом для запуска локального сервера.

1. Убедитесь, что у вас установлен Node.js.
2. Запустите в терминале команду:
   ```bash
   npm run dev
   ```
3. Откройте в браузере ссылки для просмотра вариантов дизайна:
   * **Стандартный дизайн**: [http://localhost:3000/ui_kits/pyramid-academy-web/index.html](http://localhost:3000/ui_kits/pyramid-academy-web/index.html)
   * **Глассморфизм**: [http://localhost:3000/ui_kits/pyramid-academy-glass/index.html](http://localhost:3000/ui_kits/pyramid-academy-glass/index.html)

---

### Способ 2: С помощью Python

Если у вас установлен Python, вы можете запустить сервер встроенным модулем без необходимости устанавливать дополнительные зависимости:

1. Запустите в терминале команду:
   ```bash
   python -m http.server 3000
   ```
2. Перейдите по ссылкам в браузере:
   * **Стандартный дизайн**: [http://localhost:3000/ui_kits/pyramid-academy-web/index.html](http://localhost:3000/ui_kits/pyramid-academy-web/index.html)
   * **Глассморфизм**: [http://localhost:3000/ui_kits/pyramid-academy-glass/index.html](http://localhost:3000/ui_kits/pyramid-academy-glass/index.html)

---

### Способ 3: Автономный файл (Без сервера)

В корневой директории находится файл `Pyramid Academy.html`. Он представляет собой скомпилированный автономный бандл, в который встроены все ресурсы (CSS, JS, изображения в формате Base64).
* Этот файл можно открыть **двойным кликом** непосредственно в браузере без запуска локального веб-сервера. Он будет работать полностью автономно.
