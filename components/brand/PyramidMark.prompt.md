# PyramidMark

The brand mark — three silver/navy triangles forming an upward pyramid. Inline SVG using `currentColor`, so it tints with surrounding text color.

```jsx
<PyramidMark size={40} color="var(--brand-navy-700)" />
<div style={{ color: "white" }}><PyramidMark size={56} /></div>
```

Reserve for logo placement (header, footer, hero, loading, favicon). Never as a generic decorative glyph.
