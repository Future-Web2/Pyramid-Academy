# CourseCard

**What & when:** A single course offering. Use in a 2/3-column grid on the homepage and courses page. At most one card per viewport carries `highlighted` — the featured/strategic course.

```jsx
<CourseCard
  icon={<GraduationCapIcon />}
  title="IELTS Preparation"
  description="7.0+ natijaga tayyorgarlik"
  audience="Chet elga o'qishga ketuvchilar"
  duration="3–6 oy"
  level="B2 → C1"
/>
```
