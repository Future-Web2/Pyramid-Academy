# Badge

**What & when:** Small pill labels for course duration, level chips ("A1–C2"), "Yangi guruh" callouts, status tags. Tones map directly to color semantics.

```jsx
<Badge tone="navy">Yangi guruh</Badge>
<Badge tone="accent">7.0+</Badge>
<Badge tone="neutral">3–6 oy</Badge>
```

**Tones:** `neutral` · `navy` · `accent` · `success` · `danger` · `info` · `inverse` (for navy backgrounds).
