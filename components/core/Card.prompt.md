# Card

**What & when:** Neutral container for content blocks. Use `interactive` for clickable cards (course tiles, blog posts) — gets a hover lift. Use `variant="navy"` sparingly for one promotional block per page (e.g. enrollment CTA strip).

```jsx
<Card padding="lg" interactive>
  <h3>IELTS Preparation</h3>
  <p>7.0+ natijaga tayyorgarlik</p>
</Card>
```

**Padding:** `none` · `sm` (16) · `md` (20) · `lg` (24, default) · `xl` (32).
**Variants:** `surface` · `muted` · `navy` · `outline`.
